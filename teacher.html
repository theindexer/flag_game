About

The Flag Fame is a small, 30 minute game that helps introduce you and your class to the World Peace Game. The purpose of the Flag Game is to help you, as the teacher, assess the feasibility of running the World Peace Game in your class. You can read more about the World Peace Game and the Flag Game in the documents below. After playing the game, think about the following things: Were you comfortable giving up some control to your class? Were the students able to handle the responsibility? Do you think the World Peace Game is right for your class? Be sure to browse the World Peace Game instructions before playing the Flag Game so you know what to be looking for in yourself and your students.

Instructions

There are two ways to run the flag game. The first is to use this website to run the game. After reading the instructions, you can hook up a computer to a projector and run the game on the main page. The game will help with timing rounds, generating boards, and filling in the flag. It will also generate a list of objectives for each team. You should use these objectives instead of the objectives provided in the instructions; print them out and distribute them like normal.
The second way to run the game is to print out all the materials in the instructions and draw or project a flag (like the sample above). The objectives provided in the instructions work with the sample flag provided.
Remember, do your best not to interfere! This is a mostly hands-off game.
